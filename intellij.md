# IntelliJ

## Keyboard shortcuts

### Navigation
- `⌘O` : open class
- `⇧⌘O` : open file
- `⌘E` : open recent file
- `⌘↑` : jump to navbar

### Jump to
- `⌘B` : go to declaration

### Refactor
- `⌘T` : refactor this
- `⇧F6` : rename...
- `⌥⌘V` : insert variable

### Code
- `^Enter` : generate
- `⌥Enter` : quick fix
- `⌥⌘L` : reformat file
- `⇧⌘Enter` : complete line
- `⌘D` : duplicate line
- `⇧⌥↑` : move line up
- `⇧⌥↓` : move line down
- `⌥⌘↑` : move statement up
- `⌥⌘↓` : move statement down

## Shorthands

- `psvm` : `public static void main() {`...
- `sout` : `System.out.println(...)`
